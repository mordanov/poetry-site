# One-Page Cheat Sheet — Poetry Site

## Run
```bash
cd /Users/aleksandr/Local/poetry-site
docker-compose up
```
Open `http://localhost`.

## Features
- Admin poems + drafts
- Images per poem (JPG/PNG, max 1MB)
- Version history
- Comments + moderation
- Export/import ZIP
- i18n (EN/RU/ES/FR)
