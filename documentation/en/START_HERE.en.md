# Start Here — Poetry Site

A fast orientation for the project and its documentation.

## Quick start (5 minutes)
```bash
cd /Users/aleksandr/Local/poetry-site
docker-compose up
```
Open `http://localhost`.

## What this project includes
- Admin panel for poems, drafts, images, and versions
- Public poems list with pagination and tags
- Comment system with moderation
- Export/import as ZIP (poems + comments + images)
- Multi-language UI (EN/RU/ES/FR)
