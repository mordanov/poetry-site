## 🌍 Многоязычность

✅ **Уже реализовано!**

- 🇬🇧 Английский (EN)
- 🇷🇺 Русский (RU)
- 🇪🇸 Испанский (ES)
- 🇫🇷 Французский (FR)

**Как переключить язык:**
1. Выберите язык в выпадающем списке в правом углу
2. Интерфейс переключится
3. Выбор сохраняется автоматически

**Как добавить язык:**
1. Откройте `/frontend/static/js/app.js`
2. В `I18N` добавьте все переводы для языка
3. В `/frontend/templates/index.html` добавьте опцию `<option>`
4. Готово!

Полное руководство: **ADD_NEW_LANGUAGE.md**

## 🐛 Если не работает

**Проблема:** Ошибка при `docker-compose up`
```bash
# Решение:
docker-compose down -v
docker-compose up --build
```

**Проблема:** Порт 80 занят
```bash
# Измените в docker-compose.yml:
ports:
  - "8080:80"  # Используйте 8080
  - "8443:443"
# Откройте: http://localhost:8080
```

**Проблема:** Забыли пароль админа
```bash
# Удалите БД:
docker-compose down -v

# Создайте .env:
echo "ADMIN_PASSWORD=newpass123" > .env

# Запустите:
docker-compose up
```

**Проверьте логи:**
```bash
docker-compose logs | grep -i error
```
