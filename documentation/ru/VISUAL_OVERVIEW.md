## 🎯 Поток работы: Создание стихотворения

```
┌─────────────────┐
│ 👤 ПОЛЬЗОВАТЕЛЬ │
└────────┬────────┘
         │
         │ 1. Нажимает "Admin"
         ▼
┌─────────────────┐
│ 🔐 LOGIN MODAL  │
│ Username: admin │
│ Password: ****  │
└────────┬────────┘
         │
         │ 2. POST /api/auth/login
         ▼
┌─────────────────────────────┐
│ 🐍 BACKEND                  │
│ • Проверяет пароль (bcrypt) │
│ • Генерирует JWT токен      │
│ • Возвращает токен          │
└────────┬────────────────────┘
         │
         │ 3. Сохраняет токен в localStorage
         ▼
┌──────────────────────────────┐
│ 👤 АДМИН-ПАНЕЛЬ              │
│ • Видит форму "Новое стихо"  │
│ • Вводит текст и теги        │
└────────┬─────────────────────┘
         │
         │ 4. POST /api/poems (с токеном)
         ▼
┌────────────────────────────────────┐
│ 🐍 BACKEND                         │
│ • Проверяет JWT токен (valid?)     │
│ • Вставляет стих в БД              │
│ • Создаёт теги и связи             │
│ • Возвращает новый стих            │
└────────┬───────────────────────────┘
         │
         │ 5. Показывает "Стих опубликован"
         ▼
┌──────────────────────────────┐
│ 👤 ГЛАВНАЯ СТРАНИЦА          │
│ Новое стихотворение видно ✅  │
│ (автоматически загружено)    │
└──────────────────────────────┘
```

## 🌍 Поток работы: Переключение языка

```
┌─────────────────────────────────┐
│ 👤 БРАУЗЕР (http://localhost)   │
│ Язык: English (EN)              │
│ localStorage['lang'] = 'en'     │
└────────────┬────────────────────┘
             │
             │ 1. Выбирает язык "RU" из списка
             ▼
┌──────────────────────────────┐
│ 📱 JAVASCRIPT (app.js)       │
│ setLanguage('ru')            │
│ 1. currentLang = 'ru'        │
│ 2. localStorage['lang']='ru' │
│ 3. applyTranslations()       │
│ 4. updateLangUI()            │
│ 5. handleRoute()             │
└────────────┬────────────────┘
             │
             │ 2. Обновляет все элементы
             ▼
┌────────────────────────────────┐
│ 🌐 БРАУЗЕР (ПЕРЕВЕДЕНО)        │
│ "Home" → "Главная"            │
│ "Poems" → "Стихи"             │
│ "Admin" → "Админ"             │
│ localStorage['lang'] = 'ru' ✅  │
└────────────────────────────────┘
             │
             │ 3. При перезагрузке:
             │    (F5 или Ctrl+R)
             ▼
┌────────────────────────────────┐
│ 📱 JAVASCRIPT                  │
│ currentLang = localStorage['lang']
│            = 'ru'              │
│ Применяет русский язык!        │
└────────────────────────────────┘
```
